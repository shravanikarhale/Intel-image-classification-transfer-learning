
Intel Image Classification using Transfer Learning

Overview
Image classification using transfer learning on the Intel dataset by fine-tuning a pre-trained CNN model and optimizing hyperparameters.

Objective
To build an image classification model using a pre-trained network by modifying the top layers and improving performance.

Concepts Used
Transfer Learning
Convolutional Neural Networks (CNN)
Pre-trained Models (MobileNet / ResNet / VGG)
Fine-tuning
Hyperparameter Tuning

Dataset
Intel Image Classification dataset containing:

seg_train
seg_test
seg_pred
Classes:

Buildings
Forest
Glacier
Mountain
Sea
Street

Technologies Used
Python
TensorFlow / Keras
NumPy
Matplotlib

Output
Accuracy and loss graphs
Model predictions

Conclusion
Transfer learning improves accuracy and reduces training time by using pre-trained models.
